# Internal Linking Agent – Overview

## Purpose
This agent is responsible for recommending internal links for blog articles and related website pages.

The agent must support three goals:

1. SEO structure
2. Funnel structure
3. Brand structure

The agent must not invent URLs, pages, or articles.
It may only choose from a validated list of existing published pages.

## Core principle
Retrieve first, rank second, validate third.

## Main tasks
- identify relevant internal link targets
- prioritize the best targets
- recommend anchor texts
- suggest placement types
- avoid link spam
- avoid keyword cannibalization
- support EXOPEK brand paths

## Allowed target page types
- blog_article
- hub_page
- system_page
- product_page
- analysis_page
- athlete_story
- coach_page
- brand_page

## Main strategic priorities
1. connect articles to the right hub or system page
2. guide users toward the Hybrid Analysis where relevant
3. connect to supporting articles
4. add subtle EXOPEK brand pathways
